## Scope

This documentation describes the current Phase 1 shape of AgentOS:

- one assistant identity
- one operator-only web surface
- one governed mission skill
- one bounded worker runtime
- one Postgres-backed kernel as canonical state

It does not describe:

- multi-user RBAC
- public chat-channel integrations
- generalized workflow authoring
- external runtime adapters such as OpenClaw
